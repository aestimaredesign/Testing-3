How to Use This Calculator

  1. Start by filling out the Project Input section in the General tab with your project details.
  2. Enter Wall Area (mÂ²) and Box Density (boxes per square meter) to calculate the total number of boxes needed.
  3. Go to the Circuit Types tab to define your multi-wire circuits. Add each circuit type with its wire size, percentage distribution, and NEC requirements.
  4. In the Box Distribution tab, specify what percentage of boxes should be each type (junction, utility, square, etc.). The total must equal 100%.
  5. Configure all cost parameters in the Cost Parameters tab, including labor rates, material costs, and indirect cost percentages.
  6. Click the Calculate button to generate your estimate. Review the results in the right panel.
  7. Use the Export PDF or Export Excel buttons to save your estimate for reporting.

Standard Box Sizes & NEC Compliance

Common Electrical Box Capacities

  • Utility Box (Standard): 18.0 cubic inches
  • Utility Box (Deep): 22.5 cubic inches
  • 4x4 Junction Box (1.5"): 21.0 cubic inches
  • 4x4 Junction Box (2.125"): 30.3 cubic inches
  • 4x4 Square Box (1.5"): 21.0 cubic inches
  • 4x4 Square Box (2.125"): 30.3 cubic inches
  • Two Gang Box (Standard): 30.0 cubic inches
  • Two Gang Box (Deep): 42.0 cubic inches

NEC 314.16 Key Requirements

  • Each current-carrying conductor counts as one volume unit
  • All grounding conductors combined count as one volume unit
  • Each device (switch or receptacle) counts as two volume units
  • Each internal cable clamp counts as one volume unit
  • Pigtails are counted as additional conductors
  • A 20% safety factor is recommended for engineering margin
